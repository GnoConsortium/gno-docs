GNO-Related Web Links

The following links are also likely to be of interest to GNO users and programmers.
  • The Describe Database Page. This is a database containing information on current GNO and ORCA/Shell utilities, including version numbers and from where they can be obtained.
  • Procyon Enterprises Inc
  • The Byte Works, creators of ORCA/C, ORCA/M, and other programming tools. (See their morgue for the Apple II related things.)
  • Soenke Behrens' ORCA/C Bug List
  • GS/TCP
  • Marinetti

The following links have a less direct relevance to GNO, but may still be useful:

  • The File System Hierarchy "standard". This is not a true standard in the ANSI/ISO sense, but still contains a lot of useful information. NOTE: GNO is based on 4.4BSD which does not, at this time, completely follow the suggested hierarchy. GNO sites should use the hierarchy specified in the FAQ.
